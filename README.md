# Terminal Portfolio

A terminal-style interactive portfolio built with HTML, CSS, and JavaScript.

Live demo: portofolio-ruddy-beta.vercel.app

## Features

* Interactive command-based navigation (`help`, `ls`, `cat`, etc.)
* Project explorer with real GitHub repos
* Custom terminal UI (boot sequence, matrix effect, animations)
* Lightweight and fully client-side

## Commands

Try:

* `help`
* `whoami`
* `ls`
* `cat README.md`
* `skills`
* `open [project]`


## Tech

* HTML / CSS
* Vanilla JavaScript
* Three.js

## Note

Built as a personal portfolio — focused on simplicity, interaction, and real projects over flashy marketing.
